# reversed (optional)

```js
type reversed = boolean;
```
<br />
You can reverse the chessboard to make it more comfortable for playing as black pieces.

### Code
```tsx
<ChessBoard 
    FEN="rnbqkbnr/pppppppp/8/8/8/8/PPPPPPPP/RNBQKBNR w KQkq - 0 1"
    onChange={handleChange}
    onEndGame={handleEndGame}
    reversed // <~~~ flag for reversing board
/>
```

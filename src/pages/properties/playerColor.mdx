    # playerColor

    To set the color of the piece that can only be played, the player's color must be selected. 
    If the *playerColor* has been selected, then the positions of the pieces 
    of the other color can only be changed by [change](/properties/change) properties.
    <br/>

    <h2 id="Typing" data-anchor>Typing</h2>
    <CopyBlock client:load>
        ```ts
        type playerColor = "white" | "black";
        ```
    </CopyBlock>

    <h2 id="Code" data-anchor>Code</h2>
    <CopyBlock client:load>
        ```jsx
        const App = () => {
            return (
                <ChessBoard 
                    FEN="rnbqkbnr/pppppppp/8/8/8/8/PPPPPPPP/RNBQKBNR w KQkq - 0 1"
                    onChange={handleChange}
                    onEndGame={handleEndGame}
                    change={change}
                    playerColor="white"
                />
            );
        }
        ```
    </CopyBlock>

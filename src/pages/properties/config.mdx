    <h2 id="ChessPiecesMap" data-anchor>ChessPiecesMap</h2>
    <CopyBlock client:load>
        ```ts
        type FigureType = 
            "pawn-white"
            | "bishop-white" 
            | "knight-white" 
            | "rook-white" 
            | "queen-white" 
            | "king-white"
            | "pawn-black" 
            | "bishop-black" 
            | "knight-black" 
            | "rook-black" 
            | "queen-black" 
            | "king-black";

        type ChessPiecesMap = {
            [key: FigureType]: (size /* from squareSize */) => JSX.Element;
        }
        ```
    </CopyBlock>
